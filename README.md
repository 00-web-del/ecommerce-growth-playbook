# 跨境电商独立站增长实验项目

这是一个模拟真实业务场景的电商产品项目，涵盖竞品分析、用户调研、功能设计与数据验证全流程。以影石Insta360独立站为分析样本，但方法论适用于各类DTC品牌。

## 项目背景
- 目标岗位：电商产品实习生
- 分析对象：影石Insta360独立站 (store.insta360.com)
- 竞品对比：大疆DJI、GoPro

## 项目结构
- `/docs`：分析报告、功能文档、AB测试报告
- `/design`：Figma原型截图
- `/assets`：项目用到的图片素材

## 我的贡献
- **竞品分析**：利用SimilarWeb和Google Trends对比三家品牌的流量结构，手动收集亚马逊用户评论，归纳出3个核心用户痛点
- **功能设计**：针对“配件兼容性差”痛点，设计“智能配件匹配助手”功能，用Google Stitch生成3个核心页面的设计图，输出完整功能设计文档
- **数据看板模拟**：用 Excel 制作 A/B 测试数据看板，通过柱状图和饼图直观展示实验组 vs 对照组的核心指标变化，证明功能预期可提升配件加购率 104.8%，降低退货率 27.5%
